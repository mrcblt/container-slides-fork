## Base images

`microsoft/nanoserver`

- Minimal base image
- Limited API Support, no PowerShell

`microsoft/windowsservercore`

- Equivalent to Windows Server Core

`microsoft/windows`

- Just announced at //build
- Bridging the gap to installed Windows
